What is Claude Channels?

Claude Channels is a new feature from Anthropic that turns Claude Code into an always-available assistant — one you can reach from apps you already use every day.

Instead of sitting at your computer to run Claude, you can now send it a message from your phone, a group chat, or any device. Claude receives your request, works on your actual codebase on your local machine, and sends back the results — all without you ever opening a terminal.

It currently works with Discord and Telegram, and is available as a research preview as of March 2026.

The End of the Middleman

For years, developers have been the bottleneck: translating what people need into prompts, and translating AI output back to the people who needed it. Claude Channels breaks that loop — teams can message Claude directly, without a developer in the chain.

How It Works

Link Claude Code to your Discord or Telegram account in just a few minutes.

2

Message

Send a task from your phone or desktop — in plain English, just like you'd ask a colleague.

3

Done

Claude works on your machine and sends back the results, right in the same conversation.

Get Started

Claude Channels is currently in research preview. To try it, you'll need Claude Code version 2.1.80 or later and a claude.ai login (API keys are not supported).

Head to the official Claude Code documentation to set up your first channel and start coding from anywhere.
